# 鬼哭の鐘 — ストーリーバイブル

このドキュメントは「鬼哭の鐘」の世界観・年表・キャラ設定の唯一の真実です。
セリフを書くとき・イベントを追加するときは必ずこのドキュメントと整合させること。

---

## 1. 世界観

### 1.1 時代と地理
- 架空の和風ファンタジー世界（中世日本〜江戸期の風俗）
- 山に囲まれた地方国。物語の舞台は南北に流れる山並みと、その麓の村々
- **幽路（ゆうろ）**：あの世とこの世の境にある霊的な層。鬼や死者の魂が留まる場所

### 1.2 力の体系
- **霊力（れいりょく）**：神社・術士・山伏が扱う、生命と魂に関わる力
- **呪い（のろい）**：強い恨みが結晶化した負の霊力。人を鬼に変えるほどの力を持つ
- **聖なる力（holy）**：幽路の存在に効く力。鐘の力もこの系統
- **霊撃（spirit）**：鬼の体は霊の力を含むため、霊撃が弱点になる種が多い

### 1.3 「鬼」の定義
- 元は人間。**呪いを浴びて体が鬼に変容した存在**
- 完全に正気を失った者・正気のまま苦しむ者・人間を恨む者が混在する
- 鬼は通常、幽路に縛られているが、封印が弱まると現世に戻れる
- 鬼の体は不死に近い。倒すと魂は幽路に戻る（消えるわけではない）

## 2. 年表

### 2.1 百年前の事件
1. この地方に大疫病が流行する
2. 村に住む術士 **黒呪（こくじゅ）** の家族（妻と娘）が病で死ぬ
3. 黒呪は「村人が我が家族を見捨てた」と恨み、長年禁じられた呪術を研究
4. 黒呪は山中の祠で**村人全員を鬼に変える呪い**を発動
5. 村人たちは体が鬼に変容。正気のまま苦しみ、村は地獄と化した
6. 当時の宮司が**鬼哭の鐘**を鋳造。鐘の音と神社の封印石で鬼たちを**幽路に封じた**
7. 黒呪本人は呪いの反動で死亡したが、**怨念だけが幽路の深層に残った**
8. 鐘は幽路の中心に安置され、長い眠りについた

### 2.2 百年の間
- 表向き「鬼の伝承」として語り継がれた
- 山伏や術士の間では言い伝えとして残るが、ほとんどの村人は事実を知らない
- 神社は各地に封印石を持ち続けるが、その役割は次第に忘れられた
- 幽路の封印は少しずつ弱まっていった

### 2.3 物語開始時（現在）
- 幽路の封印が弱まり、鬼たちが少しずつ現世に戻ってこられるようになった
- 鬼たちは**神社の封印石を破壊**し、自分たちを完全解放しようと動き出した
- ある夜、鬼たちが勇の村（封印石があった村）を襲い、封印石を破壊。村は炎に包まれた
- **勇は瓦礫の中で目覚め、物語が始まる**

## 3. キャラクター設定

### 3.1 勇（主人公）
- **年齢**：17〜18歳
- **役職**：村の若武者。代々、村の鎮守の社守の血を引く
- **家族**：父は数年前に病死。母は焼かれた村で行方不明
- **持ち物**：母から幼い頃に渡された**鈴（鈴守）** — 「これは鬼から村を守る鈴。肌身離さず持っていなさい」と言われた。実は神社の封印石の欠片で作られたもの
- **動機（序盤）**：母の安否を確認したい、なぜ村が焼かれたのか知りたい
- **動機（中盤以降）**：鬼の真実を知り、呪いを終わらせたいに変わる
- **性格**：素直、正義感が強い、考える前に動く
- **言葉遣い**：一人称「俺」、敬語は使わない、感情を素直に出す

### 3.2 朱鷺（とき）
- **年齢**：19歳
- **役職**：武士の娘。薙刀の使い手
- **家族**：父は隣村の防衛戦で鬼に斬られて死亡（その鬼は元々は人間だったが、朱鷺はそれを知らない）
- **動機（序盤）**：父の仇を討つ。鬼を一匹残らず討つ
- **動機（中盤以降）**：「父を殺した鬼も呪いの被害者かもしれない」という事実に揺れ動く
- **性格**：強気で気が強い、しかし弱い者には優しい
- **言葉遣い**：一人称「私」、勝ち気な口調、ただし内面の動揺は隠せない
- **重要**：序盤は「鬼＝悪」を強く主張するが、**中盤から徐々に揺らぐ**。決めつけ一辺倒では描かない

### 3.3 玄海（げんかい）
- **年齢**：60代
- **役職**：山伏。霊術使い
- **背景**：若い頃、師から百年前の事件を「言い伝え」として聞いていた。当初は半信半疑
- **動機**：山伏としての務めとして鬼を退ける。旅の中で言い伝えが事実だったと気づき、自分の使命を問い直す
- **性格**：落ち着いた老人、ユーモアもある、ただし時に頑固
- **言葉遣い**：一人称「ワシ」、語尾「〜じゃ」「〜のう」、年寄りらしい話し方
- **知識量**：「言い伝えとしては知っている」が「事実とは確信していない」のスタンス。廃寺の記録で初めて確信に至る

### 3.4 白（はく）
- **外見**：10代後半の少女。白い衣
- **正体**：百年前の村に住んでいた少女の魂（半霊体）
- **背景**：黒呪の呪いが発動した夜、村から逃げようとして山道で死んだ。死後、魂は幽路に取り込まれかけたが、村に残っていた**神社の鈴の力**で半霊体として現世に留まり続けた
- **記憶**：百年の間に記憶を失っていた。**勇が持つ鈴に共鳴して、少しずつ記憶を取り戻していく**
- **動機**：当初は「ついていきたい」だけ。記憶を取り戻すにつれ「呪いを終わらせる責務」を自覚する
- **性格**：おとなしく、内省的。ただし芯は強い
- **言葉遣い**：一人称「私」、丁寧で静かな口調

### 3.5 黒呪（こくじゅ）— ラスボスの正体
- 百年前の術士。鬼哭の霊（kikokuShin）の正体
- 妻と娘を疫病で失い、村人を恨んで呪いを発動した
- 自身も呪いの反動で死亡したが、怨念は幽路の深層に残り続けた
- 「恨みある限り、鬼は生まれ続ける」と憎悪を撒き続ける存在
- 現代に倒さなければ、新たな鬼が生まれ続ける

## 4. 鬼ボスたちの背景設定

各ボスは「百年前に呪われた村人」である。それぞれの背景を設定する。

### 4.1 焔鬼将（honooOniSho）— 第1章ボス
- **生前**：村の鍛冶屋。妻と娘がいた
- **呪われた時**：娘を抱いたまま鬼に変じた。娘は鬼となった父を見て泣き叫び、走って逃げた
- **百年の苦しみ**：娘の泣き顔が忘れられない。神社を壊して呪いから解放されたかった
- **倒された時**：「ようやく終われる」と涙を流す
- **現セリフの伏線**：「鬼の目から何かが零れた気がした。──涙？」

### 4.2 氷鬼将（kooruOniSho）— 第2章ボス
- **生前**：村の若い猟師。幼い娘がいた
- **呪われた時**：娘の髪飾りを握ったまま鬼に変じた。以来、その髪飾りを百年離さない
- **百年の苦しみ**：娘を二度と人間として抱きしめられない
- **動機**：勇たちを「自分のような目に遭わせたくない」と思い、「引き返せ」と警告する
- **倒された時**：「すまない…娘よ」と呟いて消える
- **現セリフの伏線**：「引き返せ。頼む」「子供の髪飾りが握られていた」

### 4.3 鬼大将（oniTaisho）— 第3章ボス
- **生前**：百年前の村長。村人を守る立場だった
- **呪われた時**：自分が真っ先に鬼に変じ、村人たちが続々と変容するのを見届けた
- **百年の苦しみ**：村人を守れなかった責任を背負い続けている
- **動機**：勇たちが幽路に進めば「呪いの根源」と対峙することになる。「自分たちの解放」と「勇たちの危険」の板挟みで苦しむ
- **倒された時**：「…頼む。終わりにして…くれ」（既存セリフ）
- **現セリフの伏線**：「進むな。これ以上は」「根源を断てば、我々は…」

### 4.4 幽鬼王（yuuOniO）— 第4章ボス
- **正体**：百年前に幽路に取り込まれ、最も呪いが深まった元村人の魂
- **役割**：幽路の番人と化した怨念の凝縮体
- **動機**：「もう誰にも幽路に入ってほしくない」「ここで終わらせたい」
- **倒された時**：消えゆく。鐘へと続く道が開く

### 4.5 鬼哭の霊（kikokuShin）— ラスボス
- **正体**：黒呪の怨念
- **動機**：「恨みある限り、鬼は生まれ続ける」「人間どもよ、お前たちの子孫も鬼になれ」
- **倒し方**：通常の戦闘＋鬼哭の鐘の力（kananoChikara）で完全消滅

## 5. 各章のテーマと感情曲線

### 第1章「焼かれた村→山道」：喪失と疑問
- **テーマ**：何が起きたのか分からない
- **キーシーン**：焔鬼将の涙、母の鈴の発見、朱鷺・玄海・白との合流
- **感情**：喪失感、混乱、怒り

### 第2章「湖の村→鬼の砦」：違和感の蓄積
- **テーマ**：鬼の動機が分からない
- **キーシーン**：村人の話、半壊した神社、白が「鬼が泣いていた」と言う、氷鬼将の「引き返せ」と娘の髪飾り
- **感情**：違和感、揺らぎ。朱鷺は「これは罠よ」と否定するが、内心では揺れ始める
- **重要な設計**：朱鷺の否定は最初は強く、徐々に弱まっていく

### 第3章「廃寺→幽路の入口」：真相の予感
- **テーマ**：百年前に何があったのか
- **キーシーン**：廃寺の古文書、白が半霊体だと判明、勇の鈴と白の記憶の共鳴、鬼大将の「終わりにしてくれ」
- **感情**：「鬼＝悪」の前提が崩れていく。朱鷺がついに「父を殺した鬼も…」と口にする

### 第4章「幽路の浅層→深層」：真相と覚悟
- **テーマ**：白の正体と呪いの根源
- **キーシーン**：白が記憶を完全に取り戻す、黒呪の怨念と対峙、鐘の意味を知る
- **感情**：覚悟、別れの予感

### 第5章「鐘楼」：選択
- **テーマ**：何を残し、何を失うか
- **キーシーン**：鬼哭の霊との対決、鐘を鳴らすかどうかの選択
- **感情**：A：哀しい解放／B：苦しい継続

## 7. セリフ・演出のルール

1. **鬼は単純な敵ではない**：ボス遭遇シーンには必ず「呪いの被害者」感のある描写を入れる
2. **朱鷺の否定は段階的に弱める**：第1〜2章は強く否定、第3章で揺らぎ、第4章で受容
3. **白の記憶は鈴に触れる場面で進む**：勇が鈴を取り出す／白が鈴に触れる場面を記憶トリガーにする
4. **玄海は「言い伝えとしては知っていた」スタンス**：知らないふりではなく「半信半疑だった」と表現
5. **「決めつけ」を避ける**：「絶対」「必ず」「に決まってる」を使うのは、後で覆されるキャラのセリフだけ
6. **読み手に余白を残す**：鬼の真意を全部明かさず、プレイヤーに想像させる場面を作る
